Especificación del Trabajo
Título: Proyecto Web
Descripción de Proyecto

Según se presentó en clase, deberán desarrollar un proyecto que consolida todos los conceptos aprendidos en el curso.

El objetivo es desarrollar un sitio para una organización sin fines de lucro con su propio CMS de gestión de contenidos.

En la sección Documentos/Proyectos encontrarán los archivos de base que describen el contenido de las páginas públicas.

Sobre la página de Grupo de Trabajo, existe un card para cada actividad pasada o futura. El estilo deberá distinguir entre actividades pasadas y futuras. Una actividad puede tener varios documentos relacionados

Las páginas privadas deberán ser propuestas por el grupo. Tomar en cuenta la siguiente funcionalidad requerida:

Existe un usuario administrador general y usuarios específicos que pueden editar solo los detalles de uno o varios grupos de trabajo. La creación de usuarios específicos y asignación de grupos es responsabilidad del usuario administrador general.

El CMS debe contar con autenticación de usuarios. De este modo, el usuario administrador general tiene acceso a todo el CMS, un usuario específico solo a la edición del grupo de trabajo respectivo.

 

Forma de Entrega:

ZIP en el TEC-Digital, siguiendo formato de otras entregas. Nota: incluir en el txt, las credenciales del usuario administrador general y al menos un usuario específico

Sitio publicado en el servidor web de la Escuela (no se aceptarán otros dominios)
